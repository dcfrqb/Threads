# Threads

Учебный проект **Threads** представляет собой консольное приложение на C++, демонстрирующее
основы многопоточности и синхронизации потоков.

В рамках проекта реализовано управление несколькими рабочими потоками с возможностью
их запуска, приостановки и корректного завершения выполнения.

---

## Назначение проекта

Проект предназначен для изучения:

- создания и управления потоками в C++
- работы с атомарными переменными
- синхронизации потоков
- корректного завершения многопоточных программ

---

## Функциональность

Программа создает несколько потоков, которые выполняются параллельно и управляются
пользователем через консольное меню.

Реализована возможность:

- запуска и остановки отдельных потоков
- приостановки и возобновления всех потоков
- корректного завершения программы

Вывод в консоль синхронизирован для предотвращения конфликтов между потоками.
